# 10. 향후 작업 계획 (Future Enhancements)

## 1. 단기 과제 (Short-term)

### 1.1 소셜 및 인터랙션 강화
- **댓글 시스템 (Comments)**: 작품 상세 페이지에서 유저들이 자유롭게 의견을 나누고 연출 의도를 논의할 수 있는 기능.
- **공유 기능 (Deep Link)**: 특정 작품의 시네마틱 포스터 이미지를 생성하여 소셜 미디어(Instagram, X 등)에 바로 공유할 수 있는 기능.
- **실시간 알림**: 내 작품에 티켓이 발행되거나 댓글이 달렸을 때 실시간 푸시 알림.

### 1.2 UI/UX 디테일 개선
- **무한 스크롤 (Infinite Scroll)**: 탐색 페이지에서 수백 개의 작품을 끊김 없이 볼 수 있도록 최적화.
- **검색 및 필터**: 카테고리별, 제목별, 감독(유저)별 작품 검색 기능 추가.
- **Optimistic UI 적용**: 티켓 발행 시 서버 응답을 기다리지 않고 즉시 UI를 업데이트하여 더 빠른 사용성 제공.

## 2. 중장기 과제 (Long-term)

### 2.1 시네마틱 경험 고도화
- **비디오 메모리 지원**: 사진뿐만 아니라 짧은 영상 리프레젠테이션(Reels) 업로드 및 재생 지원.

### 2.2 서비스 확장
- **시즌제 명예의 전당**: 월간/연간 단위의 시상식 페이지 구성 및 전용 배지 수여.
- **협업 제작 (Join Project)**: 하나의 목표(버킷)를 여러 명의 감독이 공동으로 달성해나가는 협업 모드.
- **모바일 앱 전환**: Capacitor 또는 React Native를 고려한 네이티브 모바일 환경 최적화.
