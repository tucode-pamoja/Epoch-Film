# 🎞️ Phase 2: Advanced Refinement & Cinematic Completion Checklist

> **Grand Chancellor's Decree**: "튼튼한 성벽과 화려한 외관을 넘어, 감독들이 실제로 '함께 영화를 찍는' 경험을 완성하고, 작은 인터랙션 하나하나에 '낭만'을 불어넣어 완성도를 매듭지어야 할 때입니다."

본 문서는 프로젝트 완성도 100% 달성을 위한 최종 고도화 작업 체크리스트입니다.

---

## 1. 🤝 협업 제작 권한 (Collaborative Editing) 보완
**목표**: 캐스팅된 멤버(Cast)가 실제 제작에 참여할 수 있도록 유기적인 권한 시스템 구축.

- [ ] **DB RLS policies 업데이트**: `memories` 테이블에 대해 캐스팅된 멤버(`is_accepted = true`)의 INSERT 권한 부여.
- [ ] **`saveMemory` 서버 액션 고도화**:
    - [ ] 작성자가 소유자(Owner)가 아닐 경우, `bucket_casts` 테이블에서 해당 사용자의 승인 상태 확인 로직 추가.
    - [ ] 캐스팅된 멤버가 추가한 메모리에는 'Special Guest' 또는 'Co-Director' 배지 표시 기능.
- [ ] **`deleteMemory` 권한 정교화**:
    - [ ] 소유자는 모든 메모리 삭제 가능.
    - [ ] 캐스팅 멤버는 자신이 추가한 메모리만 삭제 가능하도록 로직 검증.
- [ ] **캐스팅 협업 대시보드**: 버킷 상세 페이지에서 현재 참여 중인 캐스트 목록과 그들의 기여도(프레임 수) 표시.

---

## 2. ⚡ 사용자 경험(UX) 피드백 정제 (Optimistic UI)
**목표**: 네트워크 지연을 '시네마틱한 호흡'으로 승화시키거나, 즉각적인 반응성을 통해 흐름을 유지.

- [ ] **`useOptimistic` 전면 도입**:
    - [ ] 메모리 추가 시, 서버 응답 전에도 타임라인에 즉시 '현상 중...' 상태로 프레임 표시.
    - [ ] 티켓 발행(Like) 시, 즉각적으로 카운트가 오르고 별 효과가 터지는 연출.
- [ ] **프로그레시브 로딩 연출**:
    - [ ] 이미지 업로드 시, 0%에서 100%까지 차오르는 '필름 현상 바(Processing Bar)' 구현.
    - [ ] AI 로드맵 생성 시, 기계적인 로딩 스피너 대신 "시나리오 분석 중...", "촬영지 헌팅 중..." 등 대화형 시퀀스 메시지 노출.
- [ ] **에러 복구 경험**: 서버 액션 실패 시 낙관적 업데이트를 부드럽게 롤백하고, "필름이 타버렸습니다. 다시 시도해 주세요."와 같은 테마 메시지 제공.

---

## 3. 🌐 글로벌 타임라인 및 소셜 피드 최적화
**목표**: '나의 기록'을 넘어 '감독들의 네트워크'를 실시간으로 체감할 수 있는 공간 창조.

- [ ] **Premiere Feed (팔로우 피드) 고도화**:
    - [ ] Explore 페이지 내 'Following' 전용 탭 기능 강화.
    - [ ] 단순히 버킷 공유가 아닌, 팔로우한 감독의 '새로운 장면(Memory) 추가' 소식을 타임라인 형태로 노출.
- [ ] **실시간 알림(Live Notifications) 시각화**:
    - [ ] 브라우저 알림 대신 앱 내 HUD(Heads-Up Display) 상단에 "새로운 감독이 당신의 필름을 구독했습니다" 메시지 스르륵 노출.
- [ ] **활동 통계 엔진**: 내 필름의 누적 관객 수(티켓 수), 협업 횟수 등을 시각화한 'Director's Portfolio' 대시보드 강화.

---

## 4. ✨ 시네마틱 인터랙션 디테일 (Visual Polish)
**목표**: 기술을 감성으로 덮어 사용자에게 '영화 제작자'로서의 정체성 부여.

- [ ] **Focus Pulling 전환 효과**: 페이지 이동 시 배경은 흐려지고 콘텐츠는 선명해지는 심도 제어(Depth of Field) 전환 애니메이션.
- [ ] **Action Slate 애니메이션**: '제작 시작' 또는 '기록 저장' 버튼 클릭 시, 영화 슬레이트가 내려치는 소리와 시각 효과 추가.
- [ ] **Flash Bulb 피드백**: 기록 저장 성공 시 화면이 순식간에 하얗게 번쩍였다가 돌아오는 '카메라 플래시' 연출.
- [ ] **Film Grain & Light Leak**: 특정 성과 달성 시(Level Up, Quest Clear) 화면 모서리에 빈티지한 필름 노이즈와 빛 번짐 효과 적용.

---

3. **Week 3**: 시네마틱 인터랙션(슬레이트, 플래시 등) 디테일 마감 및 최종 검수.
