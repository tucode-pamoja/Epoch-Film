# EPOCH FILM (에포크 필름)

> **"당신의 시대를 기록하는 단 하나의 필름"**  
> _"Capture your epoch, Develop your dream."_

## 🎬 프로젝트 개요

EPOCH FILM은 단순한 버킷리스트 작성을 넘어, 꿈을 구체화하고 성취 과정을 영화처럼 아름답게 기록하는 **낭만 가이드 플랫폼**입니다.

### 핵심 가치 (Core Values)

- **낭만 (Romance):** 일상의 목표를 영화적 순간으로 재해석
- **성취 (Achievement):** 막연한 꿈을 현실 가능한 계획으로 전환
- **구체화 (Manifestation):** AI와 데이터를 통한 실질적 로드맵 제공
- **기록 (Archive):** 성취의 순간을 영원히 남기는 디지털 아카이브

---

## 🎞️ 주요 기능

### Part 1: 현상소 (The Lab) - 관리 및 실현

- **Archive:** 인생 전체를 아우르는 통합 버킷리스트 저장소
- **Selected Sequence:** '올해의 목표'를 선정하고 집중 관리
- **AI Director (Premium):** 현재 위치와 데이터를 기반으로 꿈 실현을 위한 최적의 로드맵 및 정보(항공, 숙박 등) 제공

### Part 2: 컷 & 셔터 (Cut & Shutter) - 기록 및 동행

- **Check-in Shot:** 단순 완료 체크가 아닌, 인증샷(사진/영상)과 타임스탬프로 성취 기록
- **Digital LP Archive:** 달성한 버킷을 음악과 함께 LP 형태의 시각적 기록으로 저장
- **Letter to Future:** 미래의 나에게 보내는 타임캡슐 편지

### Part 3: 시사회 (Premiere) - 보상 및 회고

- **Hall of Fame:** 획득한 배지와 업적을 전시하는 명예의 전당
- **Yearly Recap:** 한 해의 기록을 AI가 감성적인 영상으로 자동 편집하여 제공

---

## 🛠️ 기술 스택 (예정)

- **Frontend:** Next.js, React, Tailwind CSS
- **Backend:** Node.js / Python (FastAPI/Django)
- **Database:** PostgreSQL / Supabase
- **Storage:** AWS S3 (이미지 및 동영상 저장)
- **AI:** OpenAI API (로드맵 생성 및 콘텐츠 추천)
